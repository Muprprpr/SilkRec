# 视频编辑器使用指南

## 🎬 概述

SilkRec 视频编辑器是一个专业的视频后期处理工具，让您可以通过直观的图形界面精确控制相机运动、光标样式、背景效果等所有参数。

## ✨ 核心功能

### 1. 🎞️ 双轨道时间轴系统

#### 视频轨道
- 显示主录屏视频
- 可调整视频在画面中的占比（80%、90%、100%）
- 支持拖拽和修剪（即将支持）

#### 效果轨道
- 添加和管理视觉效果
- 支持的效果类型：
  - 📹 相机跟随效果
  - 🔍 点击缩放效果
  - ✨ 过渡效果（即将支持）
  - 🎨 颜色调整（即将支持）

#### 鼠标事件轨道（只读）
- 可视化显示所有鼠标事件
- 不同颜色标记不同事件：
  - 🔵 蓝色：左键点击
  - 🟢 绿色：右键点击
  - 🔴 红色：鼠标移动

## ⚙️ 参数控制面板

### 相机运动参数

#### 平滑强度 (0.01 - 0.5)
- **作用**：控制相机跟随的平滑程度
- **值越小**：相机运动越平滑，有延迟感
- **值越大**：相机反应越灵敏，跟随紧密
- **推荐值**：
  - 教程视频：0.10 - 0.15（默认）
  - 演示视频：0.08 - 0.12
  - 游戏视频：0.18 - 0.25

**示例效果**：
```
平滑度 = 0.05:  █░░░░░░░░ (非常平滑，电影感)
平滑度 = 0.15:  ████░░░░░ (平衡，推荐)
平滑度 = 0.30:  ████████░ (快速响应)
```

#### 缩放强度 (1.0 - 3.0x)
- **作用**：点击时的自动放大倍数
- **1.0x**：不缩放
- **1.5x**：适度放大（默认，推荐）
- **2.0x**：明显放大
- **3.0x**：极大放大（适合精细操作）

**使用建议**：
- 代码编辑：1.5x - 2.0x
- UI 演示：1.3x - 1.5x
- 精细操作：2.0x - 3.0x

#### 运动速度 (0.5 - 2.0)
- **作用**：整体相机移动速度
- **0.5x**：慢动作，适合强调
- **1.0x**：正常速度（默认）
- **2.0x**：快速，适合快节奏视频

### 视频大小调整

#### 录屏画面大小 (80% / 90% / 100%)

**80% 模式**：
```
┌─────────────────────────────┐
│        背景区域             │
│   ┌──────────────────┐      │
│   │                  │      │
│   │   录屏内容 80%   │      │
│   │                  │      │
│   └──────────────────┘      │
│        背景区域             │
└─────────────────────────────┘
```
- 适合添加装饰性背景
- 适合多画面布局
- 给背景留出空间

**100% 模式**：
```
┌─────────────────────────────┐
│                             │
│     录屏内容 100%           │
│     （全屏显示）            │
│                             │
└─────────────────────────────┘
```
- 最大化内容展示
- 适合纯内容视频
- 无背景装饰

### 🖱️ 光标设置

#### 光标样式选择

**内置样式**：
1. **默认箭头** ➡️
   - 经典Windows光标
   - 适合大多数场景

2. **手型指针** 👆
   - 适合点击演示
   - 更显眼

3. **圆形高亮** ⭕
   - 强调作用
   - 适合教学视频

4. **自定义上传** 📤
   - 支持 PNG 和 SVG 格式
   - 最大尺寸：128x128px
   - 支持透明背景

#### 光标大小 (16 - 64px)
- **16-24px**：小巧，不遮挡内容
- **32px**：标准大小（默认）
- **48-64px**：超大，适合演示

#### 点击效果
- ✅ **启用**：点击时显示涟漪动画
- ❌ **禁用**：纯净光标，无特效

---

### 🎨 背景设置

#### 背景类型

##### 1. 纯色背景
```css
background: #1a2a6c;
```
- 简洁专业
- 选择任意颜色
- 适合企业视频

##### 2. 渐变背景
```css
background: linear-gradient(135deg, #1a2a6c, #2c3e50);
```
- 现代时尚
- 支持双色渐变
- 自动计算过渡
- **推荐配色**：
  - 蓝紫渐变：`#667eea` → `#764ba2`
  - 暖橙渐变：`#f093fb` → `#f5576c`
  - 深蓝渐变：`#1a2a6c` → `#2c3e50`

##### 3. 图片背景
- 上传自定义背景图
- 支持格式：JPG, PNG, WebP
- 推荐尺寸：1920x1080 或更高
- 自动缩放和居中

**背景图片建议**：
- 使用模糊或浅色图片
- 避免复杂图案影响主内容
- 可使用品牌元素

## 🎞️ 时间轴操作

### 播放控制

- **▶️ 播放**：实时预览效果
- **⏸️ 暂停**：暂停预览
- **⏹️ 停止**：停止并重置

### 播放头操作

**拖拽播放头**：
- 鼠标拖动红色播放头
- 快速定位到任意时间点
- 实时预览该帧画面

### 效果管理

#### 添加效果
1. 点击「+ 添加效果」按钮
2. 选择效果类型
3. 拖拽调整时长和位置

#### 编辑效果
- **拖拽**：改变效果起始位置
- **拉伸边缘**：调整效果时长
- **双击**：打开详细设置（即将支持）

#### 删除效果
- 选中效果后按 Delete 键（即将支持）
- 或右键菜单选择删除

## 📤 导出设置

### 基本设置

#### 输出路径
- 默认：`output/final_export.mp4`
- 可自定义任意路径
- 自动创建目录

#### 帧率 (FPS)
- **15 fps**：低质量，小文件
- **24 fps**：电影标准
- **30 fps**：标准视频（推荐）
- **60 fps**：高刷新率，适合游戏

#### 质量选择
- **高质量**：CRF 18-20，文件较大
- **中等质量**：CRF 23，平衡（推荐）
- **低质量（快速）**：CRF 28，快速导出

#### GPU 加速
- ✅ **启用**：5-10x 速度提升（推荐）
- ❌ **禁用**：使用 CPU 编码

## 💡 使用技巧

### 1. 快速入门流程

```
1. 加载录制的视频
   ↓
2. 调整参数（平滑度、缩放）
   ↓
3. 选择光标样式和大小
   ↓
4. 设置背景（渐变/图片）
   ↓
5. 调整视频大小（80%/100%）
   ↓
6. 预览效果
   ↓
7. 导出视频
```

### 2. 参数调优策略

#### 教程视频推荐设置：
```javascript
{
  smoothness: 0.12,      // 较平滑
  zoomLevel: 1.5,        // 适度缩放
  speed: 1.0,            // 正常速度
  videoScale: 0.9,       // 90% 留出背景
  cursorSize: 32,        // 标准大小
  showClickEffect: true  // 显示点击
}
```

#### 演示视频推荐设置：
```javascript
{
  smoothness: 0.08,      // 非常平滑
  zoomLevel: 1.3,        // 轻微缩放
  speed: 0.8,            // 稍慢，更优雅
  videoScale: 1.0,       // 全屏
  cursorSize: 40,        // 较大光标
  showClickEffect: true
}
```

#### 快节奏游戏视频：
```javascript
{
  smoothness: 0.25,      // 快速响应
  zoomLevel: 1.2,        // 轻微缩放
  speed: 1.5,            // 快速移动
  videoScale: 1.0,       // 全屏
  cursorSize: 28,        // 小光标
  showClickEffect: false // 无点击效果
}
```

### 3. 背景选择指南

**根据视频类型选择**：

| 视频类型 | 推荐背景 | 说明 |
|---------|---------|------|
| 企业培训 | 纯色 | 专业、不分散注意力 |
| 产品演示 | 渐变 | 现代、有品牌感 |
| 教程视频 | 图片（浅色） | 可添加品牌元素 |
| 游戏录制 | 纯色（深色） | 不影响游戏内容 |

### 4. 性能优化建议

**导出速度优化**：
1. ✅ 启用 GPU 加速
2. 降低质量设置（如果可接受）
3. 使用较低帧率（24 fps vs 60 fps）
4. 关闭不必要的效果

**预览优化**：
1. 使用较低分辨率预览
2. 减少效果数量
3. 暂停时调整参数

## 🔧 高级功能

### 自定义光标图片

**创建自定义光标**：

1. 使用设计工具创建光标图片
2. 导出为 PNG（支持透明）
3. 推荐尺寸：32x32 到 64x64
4. 在编辑器中上传

**SVG 光标示例**：
```svg
<svg width="32" height="32" xmlns="http://www.w3.org/2000/svg">
  <circle cx="16" cy="16" r="14" fill="none" stroke="#4a9eff" stroke-width="2"/>
  <circle cx="16" cy="16" r="4" fill="#4a9eff"/>
</svg>
```

### 背景图片处理

**最佳背景图片**：
```
- 分辨率：1920x1080 或更高
- 格式：PNG（透明）或 JPG
- 处理：轻微模糊（推荐）
- 亮度：较低，避免遮盖内容
```

**Photoshop 处理步骤**：
1. 打开图片
2. 滤镜 → 模糊 → 高斯模糊（5-10px）
3. 调整 → 亮度/对比度（降低亮度 20-30%）
4. 保存为 Web 格式

## 🆘 常见问题

### Q: 预览卡顿怎么办？
**A**: 
1. 降低预览分辨率
2. 减少效果数量
3. 暂停后调整参数

### Q: 导出视频画质不好？
**A**:
1. 选择「高质量」
2. 提高帧率（30 或 60 fps）
3. 启用 GPU 加速

### Q: 背景图片不显示？
**A**:
1. 检查图片格式（支持 JPG, PNG）
2. 确保图片大小合适（< 10MB）
3. 尝试重新上传

### Q: 光标太小/太大？
**A**:
- 调整光标大小滑块（16-64px）
- 实时预览效果
- 建议：28-40px 适合大多数场景

### Q: 导出时间太长？
**A**:
1. ✅ 启用 GPU 加速（5-10x 速度提升）
2. 降低质量设置
3. 使用较低帧率（24 fps）
4. 减少视频时长
